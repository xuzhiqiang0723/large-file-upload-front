Large File Upload Pause/Resume Test

This page tests the pause/resume functionality to ensure:

  • ✓ Pause actually stops ongoing chunk uploads
  • ✓ Resume doesn't create duplicate chunk uploads
  • ✓ AbortController properly cancels XMLHttpRequest calls
  • ✓ Execution queue is properly managed

Test Instructions:

  1. Select a large file (> 10MB recommended)
  2. Calculate hash and start upload
  3. Click pause while chunks are uploading
  4. Verify that ongoing uploads stop immediately
  5. Click resume and verify no duplicate uploads occur
  6. Check browser dev tools network tab for verification

Expected Results:

  • ✓ Pausing should immediately stop all active chunk uploads
  • ✓ No new chunk uploads should start after pause
  • ✓ Resuming should only upload remaining chunks
  • ✓ No duplicate chunk uploads should occur
  • ✓ Console should show abort messages for cancelled chunks
